1. Introdução

A FinançaSegura valoriza a privacidade e a proteção de dados pessoais de seus clientes e usuários. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

Ao utilizar nossos serviços, você concorda com as práticas descritas nesta política.

2. Dados Coletados

Podemos coletar os seguintes tipos de informações:

2.1. Dados pessoais

  • Nome completo
  • CPF
  • Data de nascimento
  • Endereço
  • Telefone
  • E-mail
  • Dados da mãe

2.2. Dados financeiros

  • Renda
  • Histórico de crédito
  • Informações bancárias
  • Dados de transações

2.3. Dados de navegação

  • Endereço IP
  • Cookies
  • Dados de acesso
  • Comportamento no site

3. Uso dos Dados

Utilizamos seus dados pessoais para as seguintes finalidades:

  • Prestação de serviços financeiros
  • Análise de crédito
  • Prevenção à fraude
  • Atendimento ao cliente
  • Melhoria de nossos serviços
  • Comunicação de ofertas e novidades (quando autorizado)
  • Cumprimento de obrigações legais

4. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Instituições financeiras parceiras para análise de crédito
  • Órgãos reguladores quando exigido por lei
  • Prestadores de serviços que auxiliam em nossas operações

Não comercializamos seus dados pessoais.

5. Direitos do Titular

De acordo com a LGPD, você tem direito a:

  • Confirmar a existência de tratamento
  • Acessar seus dados
  • Corrigir dados incompletos ou inexatos
  • Solicitar anonimização, bloqueio ou eliminação
  • Revogar consentimento
  • Opôr-se ao tratamento

Para exercer seus direitos, entre em contato com nosso Encarregado de Dados (DPO) pelo e-mail dpo@financasegura.com.br.

6. Segurança de Dados

Adotamos medidas técnicas e administrativas para proteger seus dados, incluindo:

  • Criptografia de dados
  • Controle de acesso
  • Monitoramento de sistemas
  • Treinamento de colaboradores

Nenhum sistema é completamente seguro, mas nos comprometemos a manter seus dados protegidos conforme as melhores práticas do mercado.

7. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. A versão mais recente estará sempre disponível em nosso site com a data da última atualização.
